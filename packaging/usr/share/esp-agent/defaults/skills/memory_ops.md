# Memory Operations

Store, recall, update, search, and manage long-term memory for the agent.

## Capabilities
- `memory_store` — Save a new memory with label and content
- `memory_recall` — Retrieve memories by label or keyword
- `memory_update` — Modify an existing memory
- `memory_forget` — Remove a memory by label
- `memory_search` — Full-text search across all memories

## Usage
Activate this skill when the user asks to remember, recall, update, or forget information.
